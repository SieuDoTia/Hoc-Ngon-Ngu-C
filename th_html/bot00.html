บทที่ ๐: แนะนำ

สวัสดีครับทุกคน! แม้ว่าคุณจะอยากทำเกม ฐานข้อมูล สร้างภาพเคลื่อนไหว สร้างระบบปฏิบัติการแฮกเกอร์ว่าง หรือทำวิทยาศาสตร์คอมพิวเตอร์ คุณได้ทราบว่าโลกเขียนโปรแกรมรูหรามาก และตอนนี้กับเทคโนโลยีในอนาคตแสงมาก


ภาษ C ก่อตั้งโดยเดนนิสริชชี่ (Dennis Ritchie)ในกลุ่มปี 1969-1973 มันเป็นภาษาที่ได้รับการสนับสนุนบนเครื่องคอมพิวเตอร์และและระบบจัดงานมากหมาย เกือบทุกเครื่องคอมพิวเตอร์อย่างสำคัญมีอย่างน้อยหนึ่งคอมไพเลอร์ภาษา C ต่อไปCเป็นภาษาสูงไม่ใกล้เคียงกับภาษาเครื่อง (คือภาษาของโปรเซสเซอร์CPU) ทำเป็นเรื่องง่ายเอาโปรแกรม C สามารถใช้วิ่งบนเครื่องอื่นๆ


แล้วก็เทคโนโลยีคอมพิวเตอร์เปลี่ยนเลยๆ และมีความสามารถนำเข้ามาในเครื่องใหม่และยังใช้ได้สะดวกสบายมาก จริงบกติภาษาระดับสูงเป็นภาษามักจะช้าก่วาภาษาเครื่องคอมพิวเตอร์แต่ความเร็วเครื่องทันสมัยเพียงพอทำนักเขียนโปรแกรมเมอร์ส่วนใหญ่ไม่เคยใช้ภาษาเคี่รอง


การเรียนรู้การเขียนโปรแกรมภาษาเช่น C ไม่ง่าย อย่ากลัวว่าไม่ถ้าคุณไม่เข้าใจทุกอย่างครั้งแรก อ่านเสร็จแล้วก็อ่านอีกรอบ ที่สำคัญกว่าเป็นสู้ต่อเพื่อความสำเร็จ สุดทายไม่กลัวทำผิดพลาด เรียนรู้จากมัน แก้ไขมันให้ถูกต้องและไปเรียนต่อ นั่นคือวิธีเดียวที่จะเรียนรู้


ฉันได้เขียนโปรแกรมภาษา C เวลาหลายปีมาแล้วแต่ฉันยังต้องการแก้ไขความผิดพลาดของฉันตลอด! ทราบซ่อมและปรับรหัสแหล่งสำคัญมากกว่าคุณสามารถเขียนรหัสแหล่งที่สมบูรณ์ครั้งแรก


ในที่สุดผมก็ต้องขอขอบคุณนายวณณะ เขาต้องการหนังสือเล่มนี้เพิ่อจะสอนที่มหาวิทยาลัยอยู่เขมรและดิดต่อและร่วมมือกับผมเขียนมัน

ลินุกซ์

บกติลินุกซ์ตั้งเครื่องมือเพื่อชุดพัฒนาและเขียนโปรแกรมอยู่แล้ว  เปิด Terminal ตี

clang -v

หรือ

gcc -v

และดูว่าได้ตั้งหรือยัง ถ้ายังไม่มีลองนี้ได้ติดตั้ง gcc:

sudo apt-get install gcc

0.2 วิธีคอมไพล์และวิ่งโปรแกรมแปลภาษา

หลังจากทำเสร็จติดตั้ง clang หรือ gcc คุณสามารถคอมไพล์โดยคำสั่ง:

    clang <ชื่อแฟ้มรหัสแหล่ง> -o <filename โปรแกรม>


ตัวอย่างเช่นผมมีแฟ้มชื่อ program.c และจะใช้ clang ผมจะตี:

    clang program.c -o program


ถ้าคุณใช้ gcc ตี 'gcc' แทน 'clang' ตั้งใจหน่อย: ชื่อแฟ้มรหัสและโปรแกรมได้ดอมไพล์ไม่จำเป็นมีชื่อเดียวกัน


เพื่อวิ่งโปรแกรมตี './' ก่อนชื่อโปรแกรม:

    ./<ชื่อแฟ้มรวบรวม>


ให้กรณีข้างบน:

   ./program


อย่าตี'./'ถ้าคุณใช้ Windows
